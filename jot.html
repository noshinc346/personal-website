Introduction

Jot is a personalized, versatile logger application crafted specifically for Gen Z. It is designed to accommodate their unique preferences for tracking mental health, social experiences, and daily events.

Inspiration

The initial vision for Jot was to create an application that supports mental health tracking. However, we envisioned a broader scope, allowing for a space where any thought, experience, or burst of creativity could be captured.

Our app enables users to "Jot" down anything from a brilliant movie idea to a memorable or even challenging day, enhancing personal reflection and data visualization over time.

Observing that Gen Z often logs their moments using basic notes apps or uses specific apps like sobriety trackers, we were inspired to develop a more comprehensive logger app. Jot not only captures these moments but also organizes and generates insightful data automatically.

User Experience

UX / UI & Getting Started

Register Page

The registration form requires essential information only, including name, email, and password, making the sign-up process quick and straightforward. Prominent "Sign Up" and "Login" buttons guide new and returning users to the appropriate actions, enhancing user navigation. The registration page is designed to make the onboarding process as smooth as possible, allowing users to quickly access JOTâ€™s features.

Create Category

This page empowers users to create and customize categories for logging various aspects of their lives, enhancing both organization and insight. Users can create general topics such as "Study," "Dating," or any other area they wish to track. This flexibility allows for a highly personalized logging experience tailored to individual needs and interests. By organizing logs into categories and counters, users can better visualize patterns and trends in their data, facilitating deeper personal reflection and insight.

Counters

This page allows users to create distinct counters within customizable categories. Within each category, users can create specific counters to log particular activities. For example, in the "Workout" category, users can create counters like "Leg Workouts" and increment the counter each time they complete a leg workout session. Similarly, in the "Dating" category, users can set up counters for "Failed Dates" or other dating experiences. Users can tap and hold a counter button to add a detailed note about the specific count. This feature allows users to record additional information, such as thoughts, context, or specifics about the event, along with the date. This enhances the depth of logging, making it more informative and valuable over time.

Personal Data Tracker

The Data Page provides users with a visual breakdown of their logged data within each category. The pie chart displays the percentage of each counter. The Data Page is designed as a place for reflection. Users can review their logged data periodically, such as at the end of the year, to reflect on their experiences, identify trends, and gain valuable insights into their behaviors and emotions over time. The Data Page of JOT is a powerful tool that transforms logged data into meaningful insights.

What's Next?
